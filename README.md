# Scientific Calculator

A modern, feature-rich scientific calculator web application built with HTML, CSS, and JavaScript.

## Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **Scientific Functions**: 
  - Square root (√)
  - Square operation (x²)
- **Calculation History**: View all previous calculations with results
- **Dark Theme**: Built-in dark mode with theme toggle (🌙)
- **Responsive Design**: Clean and intuitive user interface
- **Error Handling**: Displays error messages for invalid operations

## Files

- `index.html` - Main HTML structure with calculator layout and buttons
- `script.js` - JavaScript logic for calculator operations and history management
- `styles.css` - CSS styling for dark theme and responsive layout

## How to Use

1. Open `index.html` in a web browser
2. Click number buttons to input values
3. Click operation buttons (+, -, ×, ÷, %) to select an operation
4. Press `=` to calculate the result
5. Use `C` to clear the display and `⌫` to delete the last digit
6. Click the moon icon (🌙) in the top-left to toggle dark/light theme
7. View calculation history below the calculator

## Operations

| Button | Function |
|--------|----------|
| `C` | Clear display |
| `⌫` | Delete last character |
| `+`, `-`, `×`, `÷`, `%` | Basic arithmetic operations |
| `x²` | Square the current value |
| `√` | Square root of the current value |
| `=` | Calculate result |

## Browser Compatibility

Works on all modern browsers that support:
- ES6 JavaScript
- CSS Grid/Flexbox
- HTML5
